# Lead Management System (Mini CRM)

A full stack Lead Management System built using React, Node.js, Express, and PostgreSQL.

## Features
Add Leads
View Leads
Update Lead Status
Delete Leads
Search Leads
Dashboard Statistics
PostgreSQL Database Integration
Responsive UI

## Tech Stack

### Frontend
React,
Vite,
Axios,
CSS

### Backend
Node.js,
Express.js,
PostgreSQL,
pg,
cors
       
## API Endpoints
### Get All Leads
GET /leads
### Add Lead
POST /leads
### Update Lead Status
PUT /leads/:id
### Delete Lead
DELETE /leads/:id
